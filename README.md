# Parcial ALSE Corte 1

Este programa en C++ calcula la distancia euclidiana entre un punto ingresado por el usuario y un conjunto de puntos.

El programa:
- Solicita el número de puntos.
- Solicita las coordenadas de cada punto.
- Solicita un punto de referencia.
- Calcula cuál es el punto más cercano.
- Muestra la distancia mínima y el índice del punto más cercano.

Compilación:
g++ main.cpp -o programa

Ejecución:
./programa
